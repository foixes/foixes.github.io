- 查看集群中各server状态、启动时间、版本等
```sql
select * from DBA_OB_SERVERS;
```

- 查看各个zone状态、IDC、Region、TYPE等信息
```sql
select * from DBA_OB_ZONES;
```

```sql
select svr_ip as RootService from DBA_OB_SERVERS where with_rootserver='yes';
```
